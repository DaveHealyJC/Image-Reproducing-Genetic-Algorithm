Mona Lisa Genetic Algorithm

Higher population gives more diversity but makes it run slower.

The number of circles each individual is made up from. More circles means a better eventual solution but makes it run slower.

0.5%-2% mutation rate is a general rule of thumb. So with 400 circles pick 2-8 mutations. 0 mutations means no diversity outside of the initial genepool and will eventually evolve into a homogenous population. (All individuals identical.)
